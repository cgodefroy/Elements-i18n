Elements 1.5.1
  • Ajout de la possibilité d'ouvrir les fichier markdown depuis d'autres applis qu'Elements.
  • Ajout de '.text' à la liste des types Markdown pour avoir une belle icone.
  • Amélioration du support d'iOS 4.2 pour les utilisateurs d'iPad.
  • Ajout de l'impression pour le sutilisateurs d'iOS 4.2.
  • No longer trying to set the parent document if the parent doesn't exist.

Elements 1.5
  • Vous pouvez désormais créer des sous dossiers pour mieux organiser vos fichiers Elements.
  • Ajout du support des fichiers formattées en Markdown (.md, .markdown, .mdown, .mdwn).
  • Ajout de la prévisualisation Markdown.
  • Ajout de l'option d'ouvrir les fichiers Markdown depuis les autres applications qui le suportent.
  • Ajout de l'envoi du contenu de fichier dans le corps du ;ail en plus de comme pièce jointe.
  • Ajout de la possibilité de créer un compte Dropbox directement depuis Elements.
  • Ajout de grilles de couleurs pour le texte et le fond dans les réglages.
  • Modification majeure de la sauvegarde de fichier. Les changements sont sauvegardées en local toutes les 5 secondes. Ils sont poussées vers Dropbox seulement quand le clavier disparaît, que l'on rafraichit la liste des fichiers ou que l'on passe l'appli en tâche de fond.
  • /Elements n'est plus codée en dur comme chemin des fichiers.
  • La performance pour les utilisateurs avec beaucoup de fichiers a été améliorée.
  • On utilise maintenant les fins de ligne Windows, afin de bien les afficher aussi sur Windows.
  • Le sujet des emails est maintenant le nom du fichier.
  • Enlevé la vue en grille sur l'iPad.
  • Amélioration de la vitesse d'affichage en vue liste.
  • Le multi tache marche mieux pour les utilisateurs iOS 4.2/iPad.
  • On force la création d'un cache s'il n'y en a pas pour un fichier.
  • Sauvegarde locale quand on reçoit une alerte de mémoire en vue édition.
  • Les snippets TextExpander sont mis à jour quand l'appli revient en premier plan.
  • Correction de l'import depuis d'autres applis.
  • Ajout de l'allemand.
